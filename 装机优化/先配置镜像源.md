## windows设置镜像源

- 路径C:\Users\知否

  - 创建pip文件夹下pip.ini

  ```
  [global]
  index-url=https://pypi.douban.com/simple
  timeout = 6000
  [install]
  trusted-host=pypi.douban.com
  disable-pip-version-check = true
  ```

## Linux设置镜像源

```
mkdir ~/.pip
vim ~/.pip/pip.conf
 
[global]
index-url=https://pypi.douban.com/simple
timeout = 6000
[install]
trusted-host=pypi.douban.com
disable-pip-version-check = true
```
